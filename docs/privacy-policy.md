# Wave Remote Privacy Policy

_Last updated: 2026-07-19_

Wave Remote controls YouTube and Netflix playback using acoustic impulses (claps
and snaps).

## Audio

All audio is captured and analyzed on your device in real time and is never
recorded, stored, or transmitted. No audio ever leaves your machine.

## Anonymous usage data

To understand how the extension is used and to improve it, Wave Remote sends
anonymous usage events to our backend (Supabase). These events contain:

- a random install identifier (not linked to you, your account, or your device),
- the event (for example: microphone permission granted, listening started, a
  playback command ran, or an error),
- the extension version and a coarse operating-system label (for example macOS,
  Windows, ChromeOS).

This data contains no audio, no page content, no browsing history, and no personal
information. It is used only to improve the product and is never sold.

## Your use-case answer

On the settings page you can tell us what you use Wave Remote for by choosing a
category or typing your own answer in the "Other" box. Your answer (including any
text you type) is sent to the same backend along with the anonymous install
identifier, so we can understand who the extension helps. Do not enter personal
information in this box.

## Feedback you submit

If you use the in-product feedback form, the message you type and an optional email
address (only if you provide one) are sent to the same backend so we can act on your
feedback.
